# 🌬️ Puff of Air

A weather web app that provides real-time weather updates for any city worldwide.
Built with **Node.js**, **Express**, and **EJS**, powered by the OpenWeather API.

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js
- **Frontend:** EJS, Bootstrap
- **API:** OpenWeather API
- **Security:** Helmet.js
